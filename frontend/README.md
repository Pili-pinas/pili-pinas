# Pili-Pinas Frontend

Streamlit UI for the Pili-Pinas voter information tool.

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
# Make sure the backend API is running first
streamlit run app.py
```

Open http://localhost:8501

## Environment

Set `PILI_PINAS_API_URL` to point to the backend (default: `http://localhost:8000`).
